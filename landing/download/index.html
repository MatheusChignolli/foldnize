Before your first real run

Copy a small sample folder or enable Dry run. Choose a renaming mode, optionally turn on Year/Month sorting, and inspect the preview. Foldnize reports files it would rename, move, or skip before you allow any change.

Local-first by design

Your media stays on your computer. Foldnize does not require an account and does not upload photos or metadata. The source code is available on GitHub under the MIT License.

Install metadata tools

Foldnize uses ExifTool to read EXIF and media metadata. Install FFmpeg to provide FFprobe as an additional fallback for MOV and MP4 creation dates. If Foldnize cannot find a valid embedded date, it skips the file instead of guessing from the filename.
